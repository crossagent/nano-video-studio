# 《骸骨之城》分镜剧本 V1

## 场景 1：晨检
**视觉描述**：小宇站在晨检仪前。背景是半透明的城市街道。晨检仪的射线扫过，小宇的四肢在屏幕上呈现出完美的、毫无秘密的青蓝色荧光，像一块洁净的果冻。
**旁白**：在骸骨之城，透明是唯一的健康标准。

## 场景 2：秘密的痛
**视觉描述**：小宇回到更衣室，拉开衣领。他的胸口有一块大约巴掌大小的、不透明的肉色皮肤。与周围透明的肋骨相比，这块皮肤显得突兀、沉重。
**旁白**：但我有一块阴影，它正变得越来越沉重。

## 场景 3：城市的真相
**视觉描述**：远景，巨大的发射塔。整座城市像是一个巨大的、被解剖的生物标本，每一辆车、每一栋楼的内部都在发光。
**旁白**：他们说这是为了绝对的公平。如果每个人都能被看穿，就没人能藏私。

## 场景 4：觉醒
**视觉描述**：小宇在奔跑。随着他的思考，他身上的实体感开始蔓延。他的脚步声不再是轻盈的“啪嗒”声，而是沉重的震动。
**旁白**：真相不是光，而是光照不透的重量。
