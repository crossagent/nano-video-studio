# 角色设定指南：小宇 (Xiao Yu)

## 1. 基础形象
*   **年龄**：15岁。
*   **体态**：清瘦，少年感。
*   **服装**：深灰色连帽衫或校服，材质轻薄。

## 2. 异化细节 (关键锚点)
*   **四肢**：完全透明，可以清晰地看到淡蓝色的骨骼骨架（不是真实的白骨，是光影形成的骸骨感）。
*   **胸口**：那是他身体的“锚点”。是一块未被穿透的实心肉体。
*   **眼神**：清澈但带着一丝疲惫和探究，他是这座城市里唯一在“观察”的人。

## 3. 动态特征
*   他在走路时，透明的部分显得很轻盈，甚至带一点漂浮感。
*   但当他产生强烈情绪时，实体部分会增加他的重力感，使他的动作变得笨拙而坚定。
