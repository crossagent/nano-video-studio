# 全局艺术风格指南 (X-Ray Dystopia)

## 1. 核心视觉关键词
*   **次表面散射 (Subsurface Scattering)**：所有生物和建筑应表现出半透明的质感。
*   **青蓝荧光 (Cyan Glow)**：代表被异化的、透明的社会能量。
*   **临床/医疗感 (Clinical Aesthetic)**：干净、冰冷、无机感。

## 2. 调色板
*   **主色**：#00FFFF (电光青) - 用于果冻质感和监控光束。
*   **背景**：#0A0A0A (深邃黑) - 用于城市阴影。
*   **强调色**：#F5DEB3 (浅象牙色) - 用于仅存的人类实体皮肤，代表温暖与人性。

## 3. 材质说明
*   **果冻态 (Gelatinous)**：高透光率，内部有微小的发光脉络。
*   **磨砂态 (Frosted)**：建筑物的质感，能看到内部结构但模糊。
*   **实体态 (Solid)**：完全不透光，表面有粗糙的纹理，与环境形成强烈反差。
