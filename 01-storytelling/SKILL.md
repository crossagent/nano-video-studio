# 故事创作 (Video Storytelling)

## 1. 技能描述
本技能是视频生产的灵魂阶段。它负责将模糊的创意转化为具有张力、逻辑自洽且引人入胜的故事。

## 2. 使用时机
- 项目启动的第一步，在编写具体脚本之前。
- 需要明确视频的核心冲突、情感基调或世界观设定时。

## 3. 执行指令
1. **核心概念定义**：明确故事的主题（Theme）和核心冲突（Conflict）。
2. **叙事结构编排**：采用三幕式结构组织情节。
3. **产出故事大纲**：编写包含起承转合的详细故事梗概。

## 4. 交付物与存放位置
- **故事大纲 (JSON)**: `01-storytelling/story_outline.json` (用于程序读取)
- **故事梗概 (Markdown)**: `01-storytelling/story_concept.md` (用于人类阅读)

## 5. 约束与规范
- 逻辑自洽，情感指向明确。
- 交付物必须严格存放在 `01-storytelling/` 目录下。

## 6. 示例
```json
{
  "title": "最后的药剂师",
  "theme": "希望与孤独",
  "logline": "在赛博朋克废墟中，最后一名药剂师试图调配出能治愈人类情感缺失的药剂。"
}
```
